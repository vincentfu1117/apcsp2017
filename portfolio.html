Scratch Remix

Partnership, September 7, 2017

Through Scratch, I've worked with my partner Derek Phung to remix an existing project called piano. We decided that using the keyboard to press the keys would be much more effecient than using the mouse to click. We added an audience applause effect and a bass to have fun. You can interact with the project through your keyboard. Spacebar to play the drum and asdfghjkwetyu for the piano. Please check out my project up above!
