Scratch Remix

Partnership, September 7, 2017

Through Scratch, I've worked with my partner Derek Phung to remix an existing project called piano. We decided that using the keyboard to press the keys would be much more effecient than using the mouse to click. We added an audience applause effect and a bass to have fun. You can interact with the project through your keyboard. Spacebar to play the drum and asdfghjkwetyu for the piano. Please check out my project up above!

Scratch Project

My boy Derek and I created a game on Scratch. This Top-Down Shooter game was inspired by our passion for the game Boxhead. Wave Defense is a game that involves using the keys wasd and left mouse button to fire shots. The goal of the game is to get the highest amount of kills. Please check out our game above!

AI2 App Inventor Project

Tilting, October 29, 2017

My boy Dan and I created a game in MIT's AI2 App Inventor called The Tilt Maze. The objective of the game is to tilt the onscreen ball sprite towards the goal without hitting any of the walls. Through the use of the accelerometer we were able to operate the tilt effect off of the tool. The game is fairly easy to play so do check it out for yourself! Click here!

Python Interactive Story

Butterfly, Novemember 5, 2017

Along with my classmates Wendy and Chloe, we created an interactive story. Heroes Offline is about a young teenager who gets invited to a VRMMORPG. Through the choices the teen makes, different outcomes would be selected. One could refer to this as the butterfly effect where each action creates a reaction. Please try out our game for yourself!
